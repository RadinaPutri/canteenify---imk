Canteenify merupakan aplikasi pemesanan makanan kantin kampus
yang dikembangkan sebagai proyek mata kuliah Interaksi Manusia dan Komputer.

## Deskripsi Proyek
Proyek ini berfokus pada perancangan UI/UX, evaluasi usability,
serta penerapan prinsip Interaksi Manusia dan Komputer.

## Arsitektur Sistem
Aplikasi Canteenify dirancang menggunakan pendekatan arsitektur
Model-View-Controller (MVC). Diagram MVC dapat dilihat pada folder diagrams.

## Tools yang Digunakan
- Figma (UI/UX Design)
- Canva (Layout & Diagram)
- Google Form (Usability Testing)
- Microsoft Excel (Rekapitulasi Data)
- GitHub (Dokumentasi Proyek)
